How to use

            
    
                
  1. 
                    1.
                    Create a config.yml file defining your plate mappings and file patterns
                
    2. 
                
  2. 
                    2.
                    ZIP your Harmony export folder containing all plate subfolders
                
    3. 
                
  3. 
                    3.
                     (Optional) Prepare a metadata.csv with "Plate" and "Well" columns. Wells must be zero-padded (e.g. A04 not A4). Any additional columns beyond these two are automatically merged into the output. Metadata is matched by Plate number and Well, and applied across all replicates.
                
    4. 
                
  4. 
                    4.
                    Upload files and click "Process Data"
                
    5. 
                
  5. 
                    5.
                    Download the concatenated CSV files
                
    6. 
            

            
            

                
Example config.yml:

                
plate_format: 384
plates:
  "P01ATBR1": {plate_number: 1, replicate: 1}
  "P01ATBR2": {plate_number: 1, replicate: 2}
  "P01ATBR3": {plate_number: 1, replicate: 3}
input_files:
  - "PlateResults.txt"
  - "Objects_Population - Nuclei.txt"
